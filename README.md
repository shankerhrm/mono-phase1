# MONO Phase-1

This repository contains the implementation and whitepaper for MONO Phase-1, a deterministic artificial cell model.

## Whitepaper

The whitepaper is available in [whitepaper/whitepaper.md](whitepaper/whitepaper.md).

To generate the PDF: `pandoc whitepaper/whitepaper.md -o whitepaper/MONO_Whitepaper_v1.0.pdf --pdf-engine=xelatex`

## Code

Run experiments with `python experiments/sweep.py`
